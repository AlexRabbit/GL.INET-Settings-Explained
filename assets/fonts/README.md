# SF Pro fonts (optional)

For best compatibility (e.g. when hosting on GitHub Pages), download the font files from the [SF Pro folder](https://github.com/bradleyhodges/SFWindows/tree/main/SF%20Pro) of the **SFWindows** repo and place them here:

- `SF-Pro-Display-Regular.otf`
- `SF-Pro-Display-Medium.otf`
- `SF-Pro-Display-Semibold.otf`
- `SF-Pro-Display-Bold.otf`
- `SF-Pro-Text-Regular.otf`
- `SF-Pro-Text-Medium.otf`
- `SF-Pro-Text-Semibold.otf`

Then the site will use these local files. If these files are not present, the CSS falls back to loading from the SFWindows repo (may be blocked by CORS on some hosts) and then to system fonts.
